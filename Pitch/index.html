Step 1

Core Loop

One sentence → playable web prototype → polished feel → shippable mobile build. Hours, not months.

✦
01
Prompt
Write the game concept in one sentence.
What to include
  • ConceptExisting game or something entirely new
  • InputMechanic, levels, win & fail conditions
  • ExtraTone, audience, platform
  • OtherAnything worth noting
Block Blast

Three tetromino pieces dealt per round. Place all three on an 8×8 grid — freely, in any order. If any piece won't fit, the round fails. Clear full rows or columns to reclaim space. Pieces grow more complex as the game progresses. Each level is a fixed number of rounds to complete.

What the AI understands
  • ConceptTetris-style grid placement puzzle
  • Input3 pieces, 8×8 grid, place all or lose. Clear rows to open space
  • ExtraComplexity scales per level. Fixed rounds to complete
  • OtherOne valid placement always guaranteed at start

Block Blast

03
Refinement
Iterate on the mechanic with the AI in the loop.
Refinement prompt

Refine the prototype: hide the grid lines, replace the white background with a richer player-friendly colour, add subtle outlines around the playfield and piece slots, trigger satisfying burst effects when rows or columns clear, and surface score gains with floating pop-ups. Every action should feel rewarding — dopamine on every placement.

Refined Block Blast

Same core loop — polished feel. Rich backdrop, outlined slots, glowing bursts on clears, floating score pops. The playfield reads like a real game.

04
Game Engine
AI rebuilds the locked mechanic in Godot or Unity, then compiles real device builds.
How it works
  • TranslateAI regenerates the refined prototype in Godot or Unity
  • BuildCompile APK for Android, IPA for iOS — one click
  • PlaytestDistribute to the team on real devices
  • IterateFeedback flows back into prompts at any stage

With the core loop locked, we're ready to layer more systems on top — economy, progression, monetization, LiveOps.

Engine prompt

Rebuild the refined web prototype as a native Godot project. Preserve the 8×8 grid, three-piece dealing, row/column clears with burst effects, floating score pops, and the dark polished visual theme. Produce a clean scene graph with reusable components, and set up export presets for Android APK and iOS IPA builds so the team can playtest on real devices.
